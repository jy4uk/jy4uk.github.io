Corona Purity Test

The Coronavirus Purity Test is based on the popular Rice Purity test. The web app lets users take a 50 question survey about their quarantine habits and activities and returns them with a score and a quarantine tier. The website has calculated over 10,000 unique tests from 9,000+ visitors.

Text 2 Sound

This program combines my passion for music with my love for coding. I used the Midi Util library to convert a user's inputed ASCII values into discrete musical notes. The program then outputs the sonified text as a MIDI file that can be directly input into a Digital Audio Workstation such as Logic Pro X.
